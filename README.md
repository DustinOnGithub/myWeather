# Weather app for android devices

* data from openweathermap.org. Using the free version. Called the 'One Call API'. YOU NEED YOUR OWN KEY
* chart.js
* Webview: JS, HTML5, CSS3
* Android: Kotlin

### features
* multiple locations
* current, daily forecast and hourly forecast weather
